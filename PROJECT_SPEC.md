Project: DomainForge-SLM



Goal:

Build and evaluate a compact domain-adaptive Small Language Model

specialized in AI/ML engineering knowledge.



Core Research Question:

Can domain-adaptive continued pretraining and parameter-efficient

instruction tuning significantly improve a compact language model's

performance on specialized AI/ML engineering tasks compared with:

1\. the original base model, and

2\. the original base model with RAG?



Target Domain:

\- Machine Learning

\- Deep Learning

\- Transformer architectures

\- Large Language Models

\- Fine-tuning and PEFT

\- Retrieval-Augmented Generation

\- LLM Evaluation

\- MLOps and model deployment



Planned Systems:

A. Base SLM

B. Base SLM + RAG

C. Domain-Adaptive SLM

D. Domain-Adaptive + Instruction-Tuned SLM

E. Domain-Adaptive + Instruction-Tuned SLM + RAG



Primary Evaluation:

\- Domain knowledge accuracy

\- Technical reasoning quality

\- Hallucination rate

\- Faithfulness to retrieved evidence

\- Retrieval Recall@K

\- Latency

\- Peak memory usage

\- Catastrophic forgetting
